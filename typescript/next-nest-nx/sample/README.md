# Sample

| Type                  | Version              |
|:---------------------|:-----------------|
| `OS`                 | maybe Linux |
| `Node.js`            | 20.9.0 (LTS) |
| `NPX`   | 10.1.0 |
| `YARN`            | 4.4.1 |

## Install
#### YARN 버전지정
```bash
corepack enable
```
#### install
```bash
yarn install
```

## Run
#### Backend
```bash
npx nx serve backend
```

#### Web
```bash
npx nx serve web
```
